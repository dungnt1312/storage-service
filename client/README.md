# Storage Service Admin Dashboard

A React-based admin dashboard for the Storage Service backend.

## Features

- API Key authentication
- File management (list, upload, download, delete)
- Image preview
- User profile management
- API key regeneration

## Development

```bash
npm install
npm run dev
```

The development server runs on `http://localhost:3000` and proxies API requests to `http://localhost:8080`.

## Build

```bash
npm run build
```

Build output is in the `dist` directory.

## Usage

1. Start the backend server on port 8080
2. Run `npm run dev`
3. Login with your API key
